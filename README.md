### 🔍 Search Engines & Text Analysis
**Course:** Search Engines and Text Analysis &nbsp;|&nbsp; **Language:** ![Python](https://img.shields.io/badge/Python-3776AB?style=flat-square&logo=python&logoColor=white)

**🎬 Sentiment Analysis on Movie Reviews**

> Binary NLP classification on the Cornell Polarity Dataset v2.0 (2,000 reviews)

| Model | Highlight |
|-------|-----------|
| Multinomial Naïve Bayes | Best accuracy: **~83%** with POS weighting ✨ |
| Bernoulli Naïve Bayes | Strong baseline on binary features |
| Boolean Naïve Bayes | Simplified presence-based features |
| Word2Vec | Distributed word representations |

**Stack:** `scikit-learn` · `nltk` · `gensim` · `pandas`
